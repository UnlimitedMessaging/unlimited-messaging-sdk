# Contributing

This SDK is auto-generated from the [OpenAPI spec](./openapi.yaml) using [Fern](https://buildwithfern.com). Do not edit files inside `sdk/typescript/src/` or `sdk/python/unlimited_messaging/` directly — they are overwritten on every release.

## What you can contribute

- **Examples** — add new usage examples in `examples/typescript/src/` or `examples/python/src/`
- **Documentation** — improve `sdk/typescript/README.md` or `sdk/python/README.md`
- **Bug reports** — open an issue on GitHub
- **Security issues** — see [SECURITY.md](./SECURITY.md)

## Local setup

Requires Docker (for Fern).

```bash
make generate   # regenerate SDKs from openapi.yaml
make install    # install SDKs locally
```
