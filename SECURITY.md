# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this SDK, please report it privately.

**Do not open a public GitHub issue.**

Email: [contact@unlimitedmessaging.app](mailto:contact@unlimitedmessaging.app)

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and aim to release a fix within 7 days for confirmed vulnerabilities.

As a thank you, we may offer free Unlimited Messaging plans for any valuable security report.
