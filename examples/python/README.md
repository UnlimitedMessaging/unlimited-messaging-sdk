# Python Examples

Code examples for the `unlimited-messaging` package.

For full documentation, see [docs.unlimitedmessaging.app](https://docs.unlimitedmessaging.app).

## Run

```bash
pip install -r requirements.txt
API_TOKEN=your_token python3 src/send_message.py
API_TOKEN=your_token python3 src/list_messages.py
API_TOKEN=your_token python3 src/list_sims.py
```

Get your API token at [app.unlimitedmessaging.app](https://app.unlimitedmessaging.app).
