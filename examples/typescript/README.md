# TypeScript Examples

Code examples for the `@unlimited-messaging/sdk` package.

For full documentation, see [docs.unlimitedmessaging.app](https://docs.unlimitedmessaging.app).

## Run

```bash
npm install
API_TOKEN=your_token npm run send-message
API_TOKEN=your_token npm run list-messages
API_TOKEN=your_token npm run list-sims
```

Get your API token at [app.unlimitedmessaging.app](https://app.unlimitedmessaging.app).
